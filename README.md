## 🚀 À propos du projet
 
Ce projet consiste en la conception d'une **page de profil utilisateur** complète en HTML5 structuré, sémantique et accessible.
 
Il a été réalisé dans le cadre du workshop **HTML Masterclass** organisé par IRIS Junior Création, en appliquant l'ensemble des bonnes pratiques vues lors des sessions : sémantique HTML5, accessibilité ARIA, validation W3C et workflow Git/GitHub professionnel.

| Technologie | Rôle |
|---|---|
| **HTML5** | Structure sémantique de la page |
| **Git** | Versionnement local et historique des commits |
| **GitHub** | Hébergement du dépôt et workflow Pull Request |
 
---
 
## ✨ Fonctionnalités
 
| Section | Description |
|---|---|
| 🧑 **En-tête** | Avatar avec `<figure>` / `<figcaption>`, nom, rôle et navigation interne par ancres |
| 📖 **À propos** | Biographie structurée avec `<section>` et `<p>` |
| 💡 **Compétences** | Liste de compétences techniques avec `<ul>` / `<li>` |
| 📬 **Contact** | Formulaire complet avec `<fieldset>`, `<legend>`, 5 types d'inputs et un `<select>` |
| 🔗 **Pied de page** | Liens GitHub, LinkedIn, Email et mention de copyright |
| ♿ **Accessibilité** | Attributs `alt`, `aria-label`, `aria-labelledby`, `lang="fr"`, hiérarchie H1 → H2 |
